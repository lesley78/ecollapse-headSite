Полезные ресурсы

Организации-партнеры

  • Московский Политехнический Университет - образовательное учреждение, в рамках которого разрабатывается проект.
  • Unity Technologies - разработчик игрового движка, используемого в проекте.

Техническая документация

  • Unity Documentation - официальная документация по игровому движку Unity.
  • C# Documentation - документация по языку программирования C#.
  • Git Documentation - документация по системе контроля версий Git.

Статьи и материалы

  • Gamasutra - ресурс с статьями о разработке игр.
  • Procedural Generation in Games - блог о процедурной генерации в играх.
  • Red Blob Games - статьи об алгоритмах для игр, включая процедурную генерацию.

Вдохновляющие проекты

  • Minecraft - классическая игра с процедурной генерацией миров.
  • No Man's Sky - игра с процедурной генерацией планет и экосистем.
  • ARK: Survival Evolved - survival-игра с элементами эволюции и приручения существ.
  • ECO - игра с динамической экосистемой, реагирующей на действия игроков.
